# Flux AI Pro 2.0

基於 Cloudflare Workers 的 AI 圖像生成服務

## 功能

- ✅ 多模型支持（Z-Image, Flux, Turbo, Kontext）
- ✅ 45+ 風格預設
- ✅ 智能 HD 優化
- ✅ 圖生圖功能
- ✅ 中文自動翻譯
- ✅ 純 Worker 架構（無需建置）

## 部署

```bash
# 設置 API Key
wrangler secret put POLLINATIONS_API_KEY

# 部署
wrangler deploy
```

## 使用

訪問 Worker URL 即可使用 Web UI 界面。
